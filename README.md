# weatherApp

## Steps to run the app:

What do you need to start the app:

- node, npm
- mongo server

In weatherApp folder:

- run `npm install`
- run `npm start`

In server folder:

- run `npm install`
- run `npm start`

If you want to watch BE, run this command:

- `npm run watch`

Notes:

First logged in user will be admin
